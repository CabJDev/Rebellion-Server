# Rebellion Server

## Overview
This is a game that was made for my capstone project at MacEwan University. The main goal was to figure out a way to develop a game similar to the "Jackbox Party Pack" games. Players are able to use a web browser to join a lobby with a provided code and control the actions of their character.

This is the web server side of the project. The server handles web-to-game communication bidirectionally. Theoretically, this set up allows for real-time games with a similar premise to the Jackbox Party Pack games.
